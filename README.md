# Universal BIOS Recovery Image for Alder Lake-N (N100)

Этот репозиторий содержит "Золотой Дамп" (Gold Dump) BIOS для ноутбуков на базе процессора Intel N100 (Alder Lake-N), полностью пересобранный для исправления структурных ошибок.

## Характеристики образа (v2.0):
*   **Регион ME:** Версия **16.50.0.1120**. Состояние: **Initialized** (авто-конфигурация при первом старте).
*   **Descriptor:** Полностью разблокирован (Unlocked). Исправлены смещения регионов (нет наложений). Допускает прошивку через `flashrom`.
*   **Сборка:** Произведена через Intel Modular FIT (mFIT) v16.50 для обеспечения идеальной целостности структуры.
*   **DMI Данные:** Очищены. Серийный номер: `SERIAL-NUMBR`.
*   **Анонимность:** Удалены UUID и ключи Windows (MSDM).

## Кому это нужно?
1. Тем, у кого "окирпичился" ноутбук.
2. Тем, у кого проблемы с ME (выключения через 30 мин).
3. Исследователям, которым нужна чистая база без наложений регионов.

## Как использовать:
1. Прошейте `bin/Gold_Final_v2.bin`.
2. После загрузки восстановите серийник: `AMIDEWINx64.exe /SS "ВАШ_СЕРИЙНИК"`
